HyperCPU Project

The hyper experimental project of custom ISA and its toolkit.


Welcome!

You have reached the official homepage of HyperCPU Project, which provides an experimental custom ISA (instruction set architecture) and various tools for it.

(No, our project is not related to CPU identification utilities like CPU-Z, as you might think.)

HyperCPU Project software is free (as in freedom). You can find source code and binaries of our projects easily from "Projects" page.

HyperCPU Project is developed and maintained by volunteers. You also become a part of our community too! We welcome everyone wishing to contribute.

Questions?

Read FAQ or project documentation firstly. If it does not help, contact us.

Strongly recommended to be read.


Troubleshooting

I want to report an error/bug, what should I do?

If you have GitHub account, open project repository and create a new issue. You can also contact the developer privately. It will also be great if you tell us your operating system version and attach some screenshots or console logs of the error/bug.


Porting

Why there is no *BSD support?

HyperCPU ISA is known to fail building from sources or throw exceptions in unexpected moments under *BSD systems. Porting other HyperCPU Project utilities to *BSD systems makes no sense if HyperCPU is not available on them. Also, nobody asked us yet about it.

Why there is no Windows support?

To make a long story short: HyperCPU Project software uses GNU/Linux system API functional which may be hard or even impossible to (re-)implement using Windows API.
